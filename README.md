# 💪 FlexFit

FlexFit is a modern AI-powered Fitness & Wellness Android application built using Kotlin and Jetpack Compose 🚀

The app helps users track their daily fitness activities, monitor step count in real-time, manage workout reminders, and maintain healthy routines through a sleek and interactive UI.

---

## ✨ Features

- 🔐 Firebase Email Authentication
- 🔑 Google Sign-In Integration
- 👣 Real-time Step Counter
- 📊 Fitness Dashboard & Analytics
- ⏰ Smart Workout Reminder System
- 🔔 Push Notifications
- 📝 Workout Activity Logger
- ☁️ Firebase Firestore Database
- 🔄 Persistent Login Sessions
- 🌙 Premium Dark Theme UI
- 🎨 Smooth Compose Animations
- 📱 Responsive Modern Interface

---

## 🛠 Tech Stack

- Kotlin
- Jetpack Compose
- Material 3
- Firebase Authentication
- Firebase Firestore
- Google Sign-In API
- MVVM Architecture
- Coroutines & StateFlow
- Android Sensor API
- AlarmManager & Notifications

---

## 🧠 Architecture

FlexFit follows modern MVVM Architecture:

- UI Layer → Jetpack Compose Screens
- ViewModel Layer → State & Business Logic
- Repository Layer → Firebase Operations
- Backend Layer → Firebase Firestore
